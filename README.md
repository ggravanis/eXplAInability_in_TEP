# XAI TEP Project

Explainable Artificial Intelligence project for fault detection, classification, and interpretation using the Tennessee Eastman Process dataset.

## Status

🚧 **Pending Upload / Work in Progress**

This repository is currently being prepared. Source code, datasets, trained models, documentation, notebooks, and experimental results will be added soon.

## Overview

The XAI TEP project focuses on applying machine learning and explainable artificial intelligence methods to the Tennessee Eastman Process benchmark.

The main goal of this project is not only to detect and classify process faults, but also to understand and explain the reasoning behind model predictions.

The Tennessee Eastman Process is a widely used benchmark in chemical and process systems engineering for evaluating fault detection, process monitoring, and diagnosis methods.

## Objectives

The main objectives of this project are:

- Develop machine learning models for TEP fault detection and classification
- Analyze normal and faulty process operating conditions
- Apply explainable AI methods to interpret model decisions
- Identify important process variables contributing to specific faults
- Compare XAI model performance and interpretability
- Support transparent and reliable data-driven process monitoring
